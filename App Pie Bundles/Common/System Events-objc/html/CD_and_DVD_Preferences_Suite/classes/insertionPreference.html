Class insertionPreference

expand

Description

a specific insertion preference

  • Defined in:
    • CD and DVD Preferences Suite
  • Inherits from:
    • item (Disk-Folder-File Suite) → item (Standard Suite)

Properties

  • customApplication alias -- application to launch or activate on the insertion of media
  • customScript alias -- AppleScript to launch or activate on the insertion of media
  • insertionAction askWhatToDo / ignore / openApplication / runAScript -- action to perform on media insertion

Notes

  • Contained by:
    • CDAndDVDPreferencesObject (CD and DVD Preferences Suite)
