desktop picture settings

  • Defined in:

  • Inherits from:

Properties

  • class_ (r/o) typeClass -- The class of the object.
  • properties record -- All of the object's properties.
  • id_ (r/o) unicodeText -- the unique ID of the item
  • name unicodeText -- the name of the item
  • changeInterval shortFloat -- number of seconds to wait between changing the desktop picture
  • displayName (r/o) unicodeText -- name of display on which this desktop appears
  • picture alias -- path to file used as desktop picture
  • pictureRotation integer -- never, using interval, using login, after sleep
  • picturesFolder alias -- path to folder containing pictures for changing desktop background
  • randomOrder boolean -- turn on for random ordering of changing desktop pictures
  • translucentMenuBar boolean -- indicates whether the menu bar is translucent
